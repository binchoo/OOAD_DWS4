# Digital Watch System Simulation
## 목적

UP 방법론의 실습과 QA 과정을 경험하기 위한 교육용 프로젝트입니다.

## 팀원

 - 지우영(팀장)
 - 곽신우
 - 여찬종
 - 유경원
 - 주재빈(QA)

## 역할
 - 지우영 : 설계, 문서작성, 모델 구현
 - 곽신우 : 설계, 문서작성, 컨트롤러, 이벤트 시스템 구현
 - 여찬종 : 설계, 문서작성, 모델 구현
 - 유경원 : 설계, 문서작성, UI 구현
 - 주재빈 : 시스템 테스팅, 빌드 및 배포
